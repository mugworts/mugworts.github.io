基础

材料中的信息

  1. 描述现象
    1. 成绩–好现象
    2. 问题–坏现象
  2. 分析影响类
    1. 意义–积极影响
    2. 危害–消极影响
    3. 双刃剑–有利有弊
  3. 表明态度类
    1. 支持及其理由
    2. 反对及其理由
  4. 剖析原因类
    1. 内外因
    2. 历史、现实
    3. 经济、法律、文化
    4. 观念、管理、制度、利益驱动
  5. 对策建议类
    1. 经验
    2. 教训
    3. 呼吁
    4. 研究报告
    5. 建议
    6. 已经采取或将要采取的措施

隐藏位置

  1. 数据
  2. 案例
  3. 分析、观点

如何从材料中提炼信息？

  1. 大的原则
    1. 带着问题阅读材料
    2. 作答字数会影响要点的判断
  2. 具体的方法
    1. 主抓提示性词汇
      1. 分析影响类–积极–消极
        1. A有利于、能助推、推动B
        2. A是B的加速器、引擎、催化剂、保障
        3. A 危机、威胁到、阻碍、破坏 B
        4. A构成类B前进的障碍。
        5. A是B的绊脚石、拦路虎
      2. 剖析原因类
        1. A折射出、反映出、暴漏出B；
        2. 之所以出现A是由于B
        3. B是A的源头，A的根源在于B
        4. 究其原因有B；
        5. A的出现与B有密不可分的关系
      3. 对策建议类
        1. XXX建议、呼吁。。。
        2. 我们一定要。。
        3. 务必要。。。
        4. 要把。。摆在重要的位置
        5. 要抓住。。应该。。。
    2. 不同材料不同的阅读方法
      1. 数据型
        1. 描述一个普遍现象
        2. 支持论证一个观点
      2. 案例型
        1. 用个案来描述普遍的现象；
        2. 通过典型案例来支持论证观点
        3. 通过案例帮助我们分析原因、影响、对策
      3. 观点型
        1. 要明确这是谁的观点
        2. 明确观点是什么
        3. 明确观点的理由是什么
        4. 判断观点正确与否
          1. 中央领导观点、中央文件政策、法律本身都是正确的。
          2. 专家、网民、群众、干部不一定正确
