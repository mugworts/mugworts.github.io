在 1000 以内，除以 3 余 2，除以 7 余 3，除以 11 余 4 的数有几个？
  1. 任何两个都没有以上关系，同余、差同、和同
  2. 设 11X+4 【从大数开始】
    1. 从 0 代入简单的数：4、15、26、37
    2. 26 除以 3 余 2
    3. 得：11&3 的公倍数+26 = 33N + 26
    4. 从 0 代入：26、59、
    5. 59 除以 7 余 3
    6. 得：33&7 的公倍数+59 = 231N + 59
    7. 得到最终的公式后，从 0 代入
    8. 0-4 共 5 个数字。直接用 1000/231 的个数字比较一下就可以找有几个 N 了。

概率 🚩

  1. 概率=满足要求的数量 / 总情况数量🚩
  2. 成立的概率 = 1 - 不成功的概率
  3. 分步概率 = 满足条件的每个步骤的概率之积 – 类似 排列组合分步。
  4. 总体（分类）概率 = 满足条件的各个情况概率之和;
